*Generated from Problem Statement on: 2026-06-11*

## Summary

A web-based curriculum authoring platform that enables a single curriculum author to generate, edit, and publish Science of Reading-aligned phonics curriculum for multilingual English learners. The software is an internal production tool — the goal is to use it to produce and publish curriculum products, not to sell the software itself. AI generates structured curriculum objects; templates render those objects into publication-ready output.

## Scope

### In:
- 5-phase MVP per product spec (see Work Packages)
- Student workbook generation (vocabulary, word/phrase/sentence practice, decodable text, activities)
- Teacher manual generation (objectives, guidance, language transfer notes, common errors)
- Assessment generation (beginner and intermediate)
- Vocabulary management with image attachment
- Orthographic mapping engine
- Direct editing of all generated content
- PDF export (student workbook + teacher manual)
- Supabase Auth (email/password; schema designed for multi-tenancy)

### Out:
- Collaboration / multi-author features
- General-purpose document editing
- Google Slides export (defer past Phase 5)
- Public-facing or student-facing interfaces
- Mobile app
