**Phase 3 — AI Generation**
- Installed OpenAI SDK, created `lib/openai.ts` singleton
- Wrote `lib/prompts/curriculum.ts`: Science of Reading system prompt, GeneratedCurriculum interface, CURRICULUM_SCHEMA for structured outputs, buildUserPrompt function
- Built `/api/generate` POST route: gpt-4o-mini with json_schema strict mode, saves to all 5 DB tables

### Decisions Made

- AI model: gpt-4o-mini (cost-efficient; sufficient for structured curriculum generation)

- Page refresh on generation: `window.location.reload()` over `router.refresh()` — forces fresh server props into child client components
